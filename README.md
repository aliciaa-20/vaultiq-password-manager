# VaultIQ

### Secure Password Manager

A secure password management application built with **Python**, **Flask**, **SQLite**, **bcrypt**, and **Fernet Encryption**.

Developed as part of the **Samsung Innovation Campus (SIC) Internship Program**.

---

## Overview

VaultIQ is a secure password management application designed to help users safely store, organize, and manage their digital credentials. The application combines secure authentication, encrypted password storage, and an intuitive user interface to provide a reliable solution for password management.

The project implements modern cybersecurity practices by utilizing bcrypt for password hashing, Fernet encryption for credential protection, and SQLite for secure local data storage.

---

## Features

- Secure user registration and login
- bcrypt password hashing
- Fernet encryption for stored passwords
- Add, view, search, edit, and delete credentials
- Password generator
- Password strength analysis
- Activity logging
- Secure SQLite database integration
- Modern web interface using Flask

## Installation

### Clone the repository

```bash
git clone https://github.com/aliciaa-20/vaultiq-password-manager.git
```

### Navigate to the project directory

```bash
cd vaultiq-password-manager
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**macOS / Linux**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
